HR Emplyee Overview

Dashboard using Tableau Public

This project features a suite of dashboards designed to provide comprehensive insights into key HR metrics. The dashboards include:

  • Employee Engagement: Visualizing sentiment trends and tracking engagement scores across departments.
  • Performance Insights: Highlighting top-performing teams, identifying patterns, and pinpointing areas for growth.
  • Compensation Analysis: Breaking down salary distributions, benefits allocations, and equity considerations.
  • Employee Overview: Providing a high-level snapshot of workforce demographics, tenure, and diversity metrics.

My process began with thorough exploratory data analysis (EDA) to uncover trends, identify data quality issues, and define the KPIs that matter most. From there, I prepared and cleaned the dataset, ensuring it was ready for seamless integration into Tableau Public. Using Tableau, I connected the dataset, designed intuitive and visually appealing graphs, and brought everything together into cohesive, interactive dashboards.

Each dashboard was crafted to address specific stakeholder needs, with features like dynamic filters, drill-down capabilities, and responsive layouts that allow users to dive deeper into the data. The result is a powerful, user-friendly platform that tells a clear, data-driven story, empowering decision-makers to act with confidence.
